OpenDelays_UK Program Version 1.0  04/11/2014
_____________________________________________________
This program gather UK train data into a database using
website http://www.realtimetrains.co.uk/(real time data)
-----------------------------------------------------
      How To Use It:
- Run it with Python 2.6 and the MySQLdb module
- Set mysql database access (lines: 13 to 16)
- Create the table and column named after the program or rename those one
  (lines: 38)
- UKstation.txt file is necessary; Set the path to call him (line: 66 -getStations function)

1. upload the python program somewhere on the server or on your device
2. set the scheduler task => every days, every hours
