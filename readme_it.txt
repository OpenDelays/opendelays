OpenDelays_IT Program Version 1.0  12/11/2014
_____________________________________________________
This program gather Italian train data into a database using
website http://www.viaggiatreno.it/viaggiatrenonew/resteasy/(real time data)
-----------------------------------------------------
      How To Use It:
- Run it with Python 2.6 and the MySQLdb module
- Set mysql database access (lines: 15 to 18)
- Create the table and column named after the program or rename those one
  (lines: 39-40)
- ITstation.txt file is necessary; Set the path to call him (line: 65 -getStations function)
- ITstation.py is available to update the station list

1. upload the python program somewhere on the server or on your device
2. set the scheduler task => every days, every hours
